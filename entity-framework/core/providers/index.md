# <a name="database-providers"></a><span data-ttu-id="20ab3-102">データベース プロバイダー</span><span class="sxs-lookup"><span data-stu-id="20ab3-102">Database Providers</span></span>

<span data-ttu-id="20ab3-103">Entity Framework Core は、データベース プロバイダーと呼ばれるプラグイン ライブラリを通じて多数の異なるデータベースにアクセスできます。</span><span class="sxs-lookup"><span data-stu-id="20ab3-103">Entity Framework Core can access many different databases through plug-in libraries called database providers.</span></span>

## <a name="current-providers"></a><span data-ttu-id="20ab3-104">現在のプロバイダー</span><span class="sxs-lookup"><span data-stu-id="20ab3-104">Current providers</span></span>

> [!IMPORTANT]  
> <span data-ttu-id="20ab3-105">EF Core プロバイダーは、さまざまなソースによってビルドされます。</span><span class="sxs-lookup"><span data-stu-id="20ab3-105">EF Core providers are built by a variety of sources.</span></span> <span data-ttu-id="20ab3-106">一部のプロバイダーは、[Entity Framework Core プロジェクト](https://github.com/aspnet/EntityFrameworkCore)の一部として保守管理されていません。</span><span class="sxs-lookup"><span data-stu-id="20ab3-106">Not all providers are maintained as part of the [Entity Framework Core Project](https://github.com/aspnet/EntityFrameworkCore).</span></span> <span data-ttu-id="20ab3-107">プロバイダーの利用を検討するとき、要件に合うよう、品質、使用許諾、サポートなどを必ず評価してください。</span><span class="sxs-lookup"><span data-stu-id="20ab3-107">When considering a provider, be sure to evaluate quality, licensing, support, etc. to ensure they meet your requirements.</span></span> <span data-ttu-id="20ab3-108">また、バージョンの互換性情報の詳細について、各プロバイダーのドキュメントを確認してください。</span><span class="sxs-lookup"><span data-stu-id="20ab3-108">Also make sure you review each provider's documentation for detailed version compatibility information.</span></span>

> [!IMPORTANT]  
> <span data-ttu-id="20ab3-109">通常、EF Core プロバイダーはマイナー バージョン間では動作しますが、メジャー バージョン間では動作しません。</span><span class="sxs-lookup"><span data-stu-id="20ab3-109">EF Core providers typically work across minor versions, but not across major versions.</span></span> <span data-ttu-id="20ab3-110">たとえば、EF Core 2.1 用にリリースされたプロバイダーは EF Core 2.2 では動作しますが、EF Core 3.0 では動作しません。</span><span class="sxs-lookup"><span data-stu-id="20ab3-110">For example, a provider released for EF Core 2.1 should work with EF Core 2.2, but will not work with EF Core 3.0.</span></span>
